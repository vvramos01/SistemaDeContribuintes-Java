# Sistema de Contribuintes

Programa em Java que calcula impostos de pessoas físicas e jurídicas utilizando conceitos de herança, polimorfismo e listas. O sistema permite cadastrar múltiplos contribuintes, onde pessoas físicas têm dedução por gastos com saúde e pessoas jurídicas possuem tributação diferenciada baseada no número de funcionários. Ao final, exibe todos os impostos calculados individualmente e o total arrecadado. Desenvolvido para demonstrar na prática a aplicação de classes abstratas, métodos sobrescritos e manipulação de coleções em Java.
